# Formulário de Cadastro de Presentes 🎁

Projeto desenvolvido para coletar informações de convidados e automatizar o recebimento de dados via e-mail.

## 🚀 Tecnologias Utilizadas
* **HTML5**: Estrutura do formulário.
* **Google Apps Script**: Engine de automação para processar o envio e disparar o e-mail via Gmail.
* **GitHub Pages**: Hospedagem da interface.

## ⚙️ Como Funciona
1. O usuário preenche Nome, Telefone e o Presente escolhido.
2. Ao clicar em enviar, os dados são disparados para um endpoint do Google Apps Script.
3. O script recebe as informações e envia um e-mail formatado para `jujhon2609@gmail.com`.

## 🛠️ Como Clonar
Se quiser testar localmente:
1. Clone o repositório: `git clone https://github.com/seu-usuario/seu-repositorio.git`
2. Abra o `index.html` no seu navegador.
